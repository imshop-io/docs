# Лайки/дизлайки отзывов (в разработке)



## Запрос

### Описание формата

****

* **`commentId`** - str, id комментария
* **`value`** - лайк/дизлайк true/false

### Пример запроса

```json
{
    "commentId": "123",
    "value": true
}
```

## Ответ

### Формата ответа

* **`success`** - boolean, успех/неуспех операции

### Пример ответа

```javascript
{
    "success": true
}
```
