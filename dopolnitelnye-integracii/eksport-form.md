## Запрос

### Описание формата

* **`formId`** - id формы в системе imshop
* **`data`** - данные формы в формате ключ-значение

### Пример запроса

```javascript
{
    "formId": "form_123",
    "data": {
        "name": "ФИО",
        "email": "test@test.test",
        "phone": "79998887766"
    }
}
```

## Ответ

### Формата ответа

* **`success`** - флаг успеха

### Пример ответа

```javascript
{
    "success": true
}
```
