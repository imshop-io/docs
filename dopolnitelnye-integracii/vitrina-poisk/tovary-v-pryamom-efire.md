### Пример запроса:

```json
{
    "country": "RU"
}
```

* **`country`** - национальный домен страны, в которой идет трансляция

### Пример ответа:

```json
{

    "items": ["789887", "961551", "55598192"]

}
```

* **`items`** - список ID товаров из фида (если в фиде есть group\_id для этого товара, то берется group\_id)
