* API ключ

### Описание формата

* **`privateId`** - id/номер примерочной в вашей системе
* **`available`** - статус примерочной

### Пример запроса

```javascript
{
    "privateId": "Room_1",
    "availalbe": true
}
```

### Формата ответа

* **`success`** - флаг успеха

### Пример ответа

```javascript
{
    "success": true
}
```
